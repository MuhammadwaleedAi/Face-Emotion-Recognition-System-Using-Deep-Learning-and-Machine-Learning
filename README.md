# Face-Emotion-Recognition-System-Using-Deep-Learning-and-Machine-Learning
A Face Emotion Recognition System uses deep learning and machine learning algorithms to detect and classify human emotions from facial expressions in images or video, such as happiness, sadness, anger, surprise, and fear.
Description

This project detects human emotions from facial expressions using Deep Learning and Machine Learning.
It captures video from a webcam, detects faces, and predicts emotions like happy, sad, angry, fear, surprise, and neutral.
Features

Real-time face detection

Emotion prediction using CNN model

Webcam support

Fast and simple system

Technologies Used

OpenCV – face detection

TensorFlow / Keras – deep learning model

NumPy – data processing

Python

Files

facialemotionmodel.json → model structure

facialemotionmodel.h5 → trained model

emotion_detector.py → main program

nstallation

Install required libraries:

pip install opencv-python tensorflow keras numpy
Run Project
python emotion_detector.py

Webcam will open.

Press ESC to exit.
